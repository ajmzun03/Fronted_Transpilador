# TranspilerDocs React

Proyecto base en React + Vite + Tailwind + React Router, inspirado en tu prototipo de Figma.

## Instalar

```bash
npm install
npm run dev
```

## Rutas

- `/` Home
- `/get-started`
- `/documentation`
- `/playground`
- `/tables-errors`

## Notas

- El playground es visual por ahora.
- La tabla de errores cambia con un dropdown.
- El sidebar está hecho estilo documentación tipo MDN/TanStack.
